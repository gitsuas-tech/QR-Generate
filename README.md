# **Cara Pemasangan Menggunakan VS Code**
1. Open VS Code
2. Buka Terminal dan Install Init, Express dan Qrcode (npm init -y), (npm install express qrcode)
3. Buat Folder dengan struktur seperti ini
4. qr-generator/
.   ├── package-lock.json
.   ├── package.json
.   ├── server.js
.   └── public/
.       └── index.html
5. Jika sudah tinggal jalankan servernya menggunakan "node server.js"
